## CAG App Reviews Sentiment Analysis
**Objective** <br>
It’s humanly impossible to manually go through each online review, assign an accurate sentiment score for each, and determine the overall sentiment for the application. <br><br>
The application will assign a score based on that key phrase to help determine whether the surrounding sentiment is positive, negative, or neutral.
